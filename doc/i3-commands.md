# Working with i3 window manager

This file lines out the core commands for using i3 window manager on manjaro os. 

## Window Management

`⊞` + `enter`   → Open terminal

`⊞` + `d` → Open app launcher (dmenu/rofi)

`⊞` + `shift` + `e` → Exit i3 session

`⊞` + `shift` + `r` → Restart i3

`⊞` + `shift` + `c` → Reload config file

`⊞` + `shift` + `q` → Close focused window

`⊞` + `f` → toggle fullscreen

`⊞` + `shift` + Space → Toggle floating mode

`⊞` + `space` → Toggle focus between tiling/floating

`⊞` + `a` → Focus parent container

## Moving Focus Between Windows

`⊞` + `h` → Focus left

`⊞` + `j` → Focus down

`⊞` + `k` → Focus up

`⊞` + `l` → Focus right

`⊞` + `arrow-keys` → Focus in that direction

## Moving Windows Around

`⊞` + `shift` + `h` → Move window left

`⊞` + `shift` + `j` → Move window down

`⊞` + `shift` + `k` → Move window up

`⊞` + `shift` + `l` → Move window right

`⊞` + `shift` + Arrow Keys → Move window in that direction

## Workspace Management

`⊞` + `1–0` → Switch to workspace 1–10

`⊞` + `shift` + `1–0` → Move window to workspace 1–10

`⊞` + `tab` → Switch to previously focused workspace

## Layout Control

`⊞` + `e` → Toggle split (horizontal/vertical)

`⊞` + `s` → Stacking layout

`⊞` + `w` → Tabbed layout

`⊞` + `r` → Resize mode (then use arrows)

## Containers & Advanced

`⊞` + `v` → Split vertically

`⊞` + `b` → Toggle i3 bar

`⊞` + `shift` + `minus` → Move window to scratchpad

`⊞` + `minus` → Show scratchpad window
